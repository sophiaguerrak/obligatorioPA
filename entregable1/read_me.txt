Guia para ejecutar el trivia

Requisitos previos:
Librerías necesarias: pandas e itertools. 
Archivo CSV: El archivo CSV con las preguntas de trivia debe estar ubicado en la misma carpeta que el archivo de Python del juego.

Para ejecutar el codigo desde la terminal: navegar hasta el directorio donde se encuentra el archivo trivia.py. Una vez en el directorio correcto, ejecutar el siguiente comando para correr el juego:
python trivia.py

Para ejecutar el codigo desde un IDE: abrir el archivo trivia.py en un IDE como Visual Studio Code y correr el codigo.
